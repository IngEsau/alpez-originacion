# ALPEZ Originación MVP

Frontend demo para originación operativa con React, TypeScript, Vite, React Router, Tailwind CSS y Lucide React.

## Alcance MVP

- Login demo sin autenticación real.
- Dashboard operativo de monitoreo.
- Flujo separado de originación por `trace_id`.
- Carga y validación simulada de INE.
- Knockouts simulados.
- Solicitudes ligadas a trazas.
- Bandeja y detalle de solicitudes.
- Listado y detalle de trazas.
- Datos y servicios mock locales.

## Comandos

```bash
npm install
npm run dev
npm run build
```

No incluye backend, base de datos ni integraciones externas reales.
